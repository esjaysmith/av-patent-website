# Autonomous Driving Patent Licensing Website

This project contains the website for licensing autonomous driving patents related to camera-only end-to-end technology.

## Project Structure

- `docs/` - Project documentation and PRDs
- `website/` - Website source files
  - `assets/` - CSS, JS, images, documents, media
  - `pages/` - HTML pages (core, blog, landing)
  - `templates/` - Reusable templates
  - `config/` - Configuration files
  - `docs/` - Website-specific documentation
  - `deployment/` - Deployment scripts and configs

## Development

1. Clone the repository
2. Navigate to the website directory
3. Open files in your editor
4. Make changes and test locally

## Deployment

See deployment/ directory for deployment instructions.
